Build with `make`
Run with `./bounce-arena`

Controls:

Player 1: left/right
Player 2: a/d
Player 3: v/n
Player 4: i/p
Player 5: j/k
Player 6: -/=

Colours:

Player 1: Red
Player 2: Green
Player 3: Yellow
Player 4: Pink
Player 5: Cyan
Player 6: Orange

Your ship crashes if you hit a bullet or go off the edge.
